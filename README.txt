DESIGN
---------------------------
-fungsi jQuery tetap berjalan seperti normal tidak terpengaruh oleh penyajian tampilan.
-pada penyajian hasil belajar dibagi menjadi tiga bagian section yaitu header, main dan description
-pada section header mendefinisikan judul dan kegiatan yang dilakukan
-pada section main terdapat baris code yang sedang dijalankan dan terdapat button untuk menjalankan ulang fungsinya.
-pada section description menjelaskan tentang fungsi pada baris code yang telah dicantumkan pada section main serta 
 penjelasan lain terkait pembangunan masing-masing halaman.
